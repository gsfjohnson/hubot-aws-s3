hubot-aws-s3
============

Manage s3 via hubot.

## Installation

### Update the files to include the hubot-aws-s3 module:

#### package.json
    ...
    "dependencies": {
      ...
      "hubot-aws-s3": ">= 0.0.1"
      ...
    },
    ...

#### external-scripts.json
    [...,"hubot-aws-s3"]

Run `npm install` to install hubot-aws-s3 and dependencies.
